# Tic Tac Toe

A beginner-friendly **Tic Tac Toe** game built in Python that allows two players to compete through a command-line interface. The game automatically detects wins, validates player moves, and identifies tie games.

## Features

* Two-player gameplay (Player X vs Player O)
* Interactive command-line interface
* Automatic win detection
* Tie game detection
* Input validation
* Prevents players from selecting occupied positions
* Displays the game board after every move

## Technologies Used

* Python 3

## Python Concepts Used

* Functions
* Lists
* Loops (`while`)
* Conditional Statements (`if`, `elif`, `else`)
* Exception Handling (`try-except`)
* User Input Validation
